# 文法講義：Wh- 疑問句 (Asking Questions) — WHQA

學好 Wh- 疑問句，最重要的是掌握兩件事：**「排對字詞的順序（倒裝句型）」** 和 **「選對疑問詞」**。本講義依序說明：倒裝公式 → 虛主詞 It → 各疑問詞的問答與特殊用法 → 考前檢查。

**本講義例句格式：** 每一句例句前先用**白話說明文法規則**，再寫例句；例句下一行用 **👉** 標出**句型結構**，方便對照。

---

## 1. 疑問句句型 — 倒裝公式

排問句時，核心是**倒裝**：把 **Be 動詞** 或 **助動詞** 移到**主詞前面**。

### 公式一：句中有 Be 動詞

*   **規則：** 句子的主要結構是「主詞 + Be …」時，改問句要把 **Be 動詞** 移到 **主詞前面**，前面再加上疑問詞。

    `Where are you?` (你在哪裡？)  
    👉 疑問詞 + Be 動詞 + 主詞 …?

*   **規則：** 現在進行式是「Be + V-ing」，改問句時一樣把 **Be** 移到主詞前。

    `What is she doing?` (她在做什麼？ — 現在進行式)  
    👉 疑問詞 + Be 動詞 + 主詞 + V-ing …?

### 公式二：句中有一般動詞

*   **規則：** 句子的主要動詞是「一般動詞」（如 live, go）時，問句要借 **do/does/did** 當助動詞，助動詞移到主詞前，**主要動詞改回原形**。

    `Where does he live?` (他住在哪裡？)  
    👉 疑問詞 + 助動詞 (does) + 主詞 + 原形動詞 …?

*   **規則：** 過去式用 **did**，後面的主要動詞一律用**原形**。

    `What did you buy?` (你買了什麼？ — 過去式)  
    👉 疑問詞 + 助動詞 (did) + 主詞 + 原形動詞 …?

**📌 考試最常錯**
- 漏掉 **do/does/did** 或 **Be**（*Where you live?*、*What he wants?*）。
- **倒裝順序錯**（*Where you are?*、*What you are doing?*）。
- **助動詞後動詞未還原**（*Did you went?*、*Does he likes?*）。
- 一般動詞句誤加 **Be**（*Where are you live?*）— 應為 **Where do you live?**

## 2. 虛主詞 It

**規則：** 英文每句都要有主詞；談**時間、距離、天氣、星期**等沒有具體「誰在做動作」時，常用 **It** 當**虛主詞**（不必翻成「它」）。

*   **規則：** 問「現在幾點」用 **What time is it?**，答句用 **It** 當主詞帶出時間。

    `What time is it?` 👉 **It** is 7:00.  
    👉 問：疑問詞片語 + Be + 主詞 it?／答：It + Be + 時間

*   **規則：** 問「有多遠」常用 **How far is it?**，答句用 **It** 帶出距離。

    `How far is it?` 👉 **It** is three miles.  
    👉 問：How far + Be + it?／答：It + Be + 距離

*   **規則：** 問「要花多久」用 **How long does it take?**，答句用 **It takes …**。

    `How long does it take?` 👉 **It** takes 20 minutes.  
    👉 問：How long + does + it + take …?／答：It + takes + 時間

*   **規則：** 問天氣用 **How is the weather?**，答句常用 **It** 描述天氣狀況。

    `How is the weather?` 👉 **It** is raining.  
    👉 問：How + Be + the weather?／答：It + Be + 現在分詞／形容詞

*   **規則：** 問「今天星期幾」用 **What day is it today?**，答句用 **It is + 星期**。

    `What day is it today?` 👉 **It** is Monday.  
    👉 問：What day + Be + it …?／答：It + Be + 星期

**代名詞 It（與虛主詞不同）**

*   **規則：** 問句裡已經出現單一**物品或動物**，答句可用 **It** 代替那個名詞，避免重複。

    `Where is the cat?` 👉 **It** is under the desk. (牠在書桌下。)  
    👉 問：Where + Be + 名詞?／答：It + Be + 介係詞片語

**📌 考試最常錯**
- 該用 **It** 卻寫 **He is 7:00.** 或 **The time is rain.**（時間／天氣／距離主詞錯置）。
- **How long does it take** 答成 *It is 20 minutes.* — 花費時間應用 **It takes …**。
- **What time** 問句漏 **it**（*What time is?*）— 應為 *What time **is it**?*
- 把**虛主詞 It** 與**代名詞 it**（代替貓／書）搞混時，至少記：天氣、幾點、星期、*It takes …* 多用 **It** 開頭。

## 3. When (何時) / What time (幾點) — 問句與兩者差異

*   **規則：** 問「事件／行程在何時、幾點」且句中沒有其他一般動詞時，用 **When / What time + Be + 主詞**。

    `When is the meeting?` (會議是何時？)  
    👉 When + Be 動詞 + 主詞 …?

    `What time is the movie?` (電影是幾點？)  
    👉 What time + Be 動詞 + 主詞 …?

*   **規則：** 問「習慣、動作在何時發生」且句中有一般動詞時，用 **When / What time + do/does + 主詞 + 原形動詞**。

    `When do you wake up?` (你何時起床？)  
    👉 When + 助動詞 + 主詞 + 原形動詞 …?

    `What time do you wake up?` (你幾點起床？)  
    👉 What time + 助動詞 + 主詞 + 原形動詞 …?

*   **規則（差異）：** **When** 可問大範圍時間（幾點、星期、日期、月份、年）；**What time** 只問**幾點幾分**。

    *   **When**：範圍大 — 幾點、星期幾、日期、月份、年份皆可答。
    *   **What time**：範圍小 — 專問具體時刻。

**📌 考試最常錯**
- 選擇題：**問「幾點幾分」**卻選 **When** 當唯一答案（情境若強調「精確時刻」，**What time** 更精準）。
- **Be 句**與 **do 句**混用（*When is you wake up?* / *When do you the meeting?*）。
- 問「電影幾點」用錯助動詞（*When does the movie?*）— 無一般動詞時用 **When is …?** / **What time is …?**

## 4. 時間介係詞 (in / on / at) 用法 + When / What time 詳答例句

*   **規則：** **at** 用在「明確時刻」或固定說法（如 at night）。

    `at 7:00`、`at noon`、`at midnight`、`at night`  
    👉 at + 時刻／固定時段用語

*   **規則：** **on** 用在「特定日子」（星期、日期、節日那天、weekends）。

    `on Monday`、`on May 1st`、`on my birthday`、`on weekends`  
    👉 on + 特定日子

*   **規則：** **in** 用在「較長時段」（年、季節、月份、the morning/afternoon/evening）。

    `in 2026`、`in summer`、`in the morning` / `in the afternoon` / `in the evening`  
    👉 in + 較長時段

*   **規則：** 回答「幾點起床」這類問題，可用**介係詞片語**或**完整句**。

    *問* `What time do you wake up?` → *答* `At 7:00.` / `I wake up at 7:00.`  
    👉 答：At + 時刻／主詞 + 動詞 + at + 時刻

*   **規則：** 回答「生日在何時」用 **on + 日期**；也可用 **It's on …**。

    *問* `When is your birthday?` → *答* `On May 12th.` / `It's on May 12th.`  
    👉 答：On + 日期／It + Be + on + 日期

*   **規則：** 回答「何時讀英文」若指「在晚上這段時間」，用 **in the evening**。

    *問* `When do you study English?` → *答* `In the evening.` / `I study in the evening.`  
    👉 答：In + the evening／主詞 + 動詞 + in the evening

*   **規則：** 時間前已有 **this/that/next/last/every** 或 **today/tomorrow/yesterday** 時，**不再加** in/on/at。

    ❌ `on tomorrow` → ⭕ `tomorrow`  
    👉 時間副詞單獨使用，不加介係詞

    ❌ `in last year` → ⭕ `last year`  
    👉 last + 時間名詞 前不加 in

**📌 考試最常錯**
- **at / on / in** 選錯（*in Monday*、*on 7:00*、*at July*）。
- **at night** 與 **in the evening** 互換錯。
- **next/last/this/every + 時間** 前又多 **on/in/at**（*on tomorrow*、*in last year*）。
- 星期寫成 *at Monday* — 星期前用 **on**。

## 5. Where (哪裡) — Be／一般動詞問句 +「單字即地點」答句

*   **規則：** 問「人或物**現在在哪**」且句中沒有其他一般動詞，用 **Where + Be + 主詞**。

    `Where are you?` (你在哪裡？)  
    👉 Where + Be 動詞 + 主詞 …?

    `Where is the cat?` (貓在哪裡？)  
    👉 Where + Be 動詞 + 主詞 …?

*   **規則：** 問「住在哪、去了哪」等含一般動詞時，用 **Where + do/does/did + 主詞 + 原形動詞**。

    `Where does he live?` (他住在哪裡？)  
    👉 Where + 助動詞 + 主詞 + 原形動詞 …?

    `Where did you go yesterday?` (你昨天去哪裡？)  
    👉 Where + did + 主詞 + 原形動詞 …?

*   **規則：** **here, there, upstairs** 等是「地點副詞」，本身等於「在…地方」，**前面不加介係詞**。

    `I'm here.` / `She is there.` / `He is upstairs.` / `They are downtown.`  
    👉 主詞 + Be + 地點副詞

*   **規則：** 回答「Maria 在哪裡」若用 **inside/outside**，同樣當副詞用，常不加介係詞。

    *問* `Where is Maria?` → *答* `She is inside.` / `She is outside.`  
    👉 主詞 + Be + inside／outside

**📌 考試最常錯**
- ***Where are you live?*** — 應為 **Where do you live?** 或 **Where are you?**
- **here/there/upstairs** 前多加 **at/in**（*at here*）。
- **Where** 問句漏 **Be** 或 **do**（*Where the cat?*）。

## 6. 地方介係詞介紹

**規則：** 要說「在某一個具體地點的相對位置」時，用 **介係詞 + 名詞（片語）**。

*   **規則：** **in** 表示在封閉空間「裡面」。

    `in his room`、`in the box`  
    👉 in + 名詞

*   **規則：** **on** 表示接觸表面的「上面」，或 **on the bus** 這類慣用說法。

    `on the table`、`on the bus`  
    👉 on + 名詞

*   **規則：** **under** 表示正下方。

    `under the desk`  
    👉 under + 名詞

*   **規則：** **at** 用在「明確地點、站點」或 **at work**（在上班）。

    `at the library`、`at work`、`at the bus stop`  
    👉 at + 地點

*   **規則：** **above** 表示上方但不接觸；**next to/beside** 旁邊；**behind** 後面；**between** 兩者之間；**in front of** 前面；**across from/opposite** 對面。

    `above the box` / `next to Maria` / `behind the door` / `between the bank and the park` / `in front of the TV` / `across from the hospital`  
    👉 介係詞 + 名詞（片語）

*   **規則：** 敘述位置時，整句常為 **主詞 + Be + 介係詞片語**。

    `The cat is under the desk.` / `Bob is at the library.`  
    👉 主詞 + Be + 介係詞片語

**📌 考試最常錯**
- **on the bus** 與 **in the car**／**in a taxi** 慣用搭配寫反。
- **next to** 漏寫 **to**（*next the chair*）。
- **between** 後用錯連接（*between the bank to the park*）— 應 **between A and B**。
- **in front of** 寫成 *in front the*（漏 **of**）。

## 7. What (什麼)、What + 名詞、Which (哪一個) — 問答與 Which vs. What

*   **規則：** 問「那是什麼」用 **What + Be + 主詞**；答句可用 **It is + 名詞**。

    `What is that?` → `It is a book.`  
    👉 問：What + Be + 主詞?／答：It + Be + 名詞

*   **規則：** 問「你想要什麼」用 **What + do/does + 主詞 + 原形動詞**。

    `What do you want?` → `I want an apple.`  
    👉 問：What + 助動詞 + 主詞 + 原形動詞?／答：主詞 + 動詞 + 受詞

*   **規則：** **What + 名詞** 用來把問題問得更細（時間、顏色、種類等）。

    `What time is it?` / `What color is your car?`  
    👉 What + 名詞 + Be …?

    `What kind of shoes did you buy?` → `Running shoes.`  
    👉 What kind of + 名詞 + 助動詞 + 主詞 + 過去式動詞?／答：名詞（種類）

*   **規則：** 問「哪一個是…」且句中無一般動詞，用 **Which + Be + 主詞**（可列出選項）。

    `Which is your car, the red one or the blue one?`  
    👉 Which + Be + 主詞 …, A or B?

*   **規則：** 問「你要哪一支筆」用 **Which + 名詞 + do/does + 主詞 + 原形動詞**。

    `Which pen do you want?` → `I want the red one.`  
    👉 Which + 名詞 + 助動詞 + 主詞 + 原形動詞?

*   **規則：** **Which** 可接單複數名詞，也可用 **Which one** 問人。

    `Which earrings are yours?` / `Which one is your daughter?`  
    👉 Which (+ one) + Be …?

*   **規則（差異）：** **Which**＝選項有限；**What**＝開放式、沒限定選項。

    *   **Which**：桌上兩支筆選一支 → 用 Which。
    *   **What**：禮金想買什麼都可以 → 用 What。

**📌 考試最常錯**
- 桌上**明明二選一**卻只選 **What**（考題常要求 **Which**）。
- ***Which color …?*** — 顏色用 **What color …?**
- **What** 與 **Which** 後接名詞時**冠詞／複數**搭配錯（*Which the pen* 依題幹調整）。

## 8. What … do（各種時態的動作）& What do you do?

*   **規則：** 問「（平常）做什麼」用 **What + does + 主詞 + do**（第三人稱單數現在式）。

    `What does Bob do every morning?` → `He goes to class.`  
    👉 What + does + 主詞 + 原形動詞 do …?

*   **規則：** 問「昨天做了什麼」用 **What + did + 主詞 + do**。

    `What did you do yesterday?` → `I went downtown.`  
    👉 What + did + 主詞 + do …?

*   **規則：** 問「正在做什麼」用 **What + Be + 主詞 + doing**。

    `What is Anna doing?` → `She is studying.`  
    👉 What + Be + 主詞 + V-ing …?

*   **規則：** 問「打算做什麼」用 **What + be + 主詞 + going to + 原形**。

    `What are you going to do tomorrow?` → `I'm going to the beach.`  
    👉 What + be + 主詞 + going to + 原形動詞 …?

*   **規則：** 問「想做什麼」用 **want to / would like to**。

    `What do you want to do tonight?` / `What would you like to do tomorrow?`  
    👉 What + 助動詞 + 主詞 + want to／would like to + 原形 …?

*   **規則（職業陷阱）：** **What do you do?** 問的是**職業**，不是「你現在在做什麼」。

    `What do you do?` → `I'm a teacher.` / `I am a software engineer.`  
    👉 What + do + 主詞 + do?／答：主詞 + Be + 職稱

**📌 考試最常錯**
- **What do you do?** 答成「正在做的事」（*I am reading.*）— 題意是問**職業**。
- **What does he do** 後面又加 **ing** 或重複動詞結構錯亂。
- **did** 後用過去式（*What did you went?*）— 應 **What did you do?**／動詞原形。
- **What are you doing?** 與 **What do you do?** 題幹未看清就套錯句型。

## 9. How 搭配 Be 動詞 — 問「狀態」與回答

*   **規則：** 問「你好嗎、情況如何」用 **How + Be + 主詞**。

    `How are you?` / `How is your brother?` / `How was the trip?`  
    👉 How + Be 動詞 + 主詞 …?

*   **規則：** 回答「狀態」常用 **主詞 + Be + 形容詞**（或簡短評價）。

    `I'm fine, thanks.` / `He is busy.` / `It was great.`  
    👉 主詞 + Be + 形容詞／評價

**📌 考試最常錯**
- **How are you?** 答成交通方式（*By bus.*）— 應答**狀態／感受**。
- 形容詞前誤加 **a**（*I am a tired.*）。
- **How** 與 **What** 混淆（*What are you?* 問身分 vs *How are you?* 問近況）。

---

## 10. How 搭配一般動詞 — 問「交通方式」與回答

*   **規則：** 問「怎麼去／怎麼到」用 **How + do/does + 主詞 + 原形動詞（go, get）**。

    `How do you go to school?` / `How does she get to work?`  
    👉 How + 助動詞 + 主詞 + 原形動詞 …?

*   **答句：**

     👉On = 可站立、行走、大空間 
     
     On the bus, On the train, On the plane
     
     👉In = 需縮身、小空間 
     
     In the car, In the taxi, In the truck
     
     👉By = 交通方式，前面不用加冠詞 
     
     By taxi, By air

**📌 考試最常錯**
- **by bus** 中間加 **a/the**（*by a bus*）。
- 走路寫成 *by foot* — 應 **on foot**。
- **How do you go …?** 與 **Where do you go …?** 題意未分就選錯疑問詞。

## 11. How often（多常／頻率）— Be 與一般動詞問句與答句

*   **規則：** 問「多常做某動作」用 **How often + do/does + 主詞 + 原形動詞**。

    `How often do you play tennis?` → `Twice a week.` / `I usually play twice a week.`  
    👉 How often + 助動詞 + 主詞 + 原形動詞 …?

*   **規則：** 問「多常在某一狀態／地點」可用 **How often + Be + 主詞 + 補語／介係詞片語**。

    `How often are you at the library?` → `Three times a week.`  
    👉 How often + Be + 主詞 + at + 地點 …?

    `How often was he sick last month?` → `He was sick twice.`  
    👉 How often + Be + 主詞 + 形容詞 …?

*   **規則：** **頻率副詞** 放在 **Be 動詞後面**、**一般動詞前面**。

    `I usually play tennis.`  
    👉 主詞 + 頻率副詞 + 一般動詞 …

*   **規則：** **次數片語**（如 once a week）常放在**句尾**。

    `I play tennis twice a week.`  
    👉 主詞 + 動詞 … + 次數片語

**📌 考試最常錯**
- **頻率副詞**放錯位置（*I play usually tennis.*）— 應 **usually** 在一般動詞前；**Be** 後則 *He is usually …*。
- **How often** 與 **How long** 選錯（一個問「多常」，一個問「多久」）。
- 答 **How often** 卻只寫形容詞而無頻率或次數（*Often.* 可接受時態要與題幹一致）。

## 12. How long（多久／時間長度）— Be 與一般動詞問句與答句

*   **規則：** 問「電影、活動持續多久／多長」用 **How long + Be + 主詞 …**。

    `How long is the movie?` → `It's two hours.` / `The movie is two hours long.`  
    👉 How long + Be + 主詞 …?

    `How long will you be in Hong Kong?` → `Ten days.`  
    👉 How long + will + 主詞 + be …?

*   **規則：** 問「做了多久、讀了多久」用 **How long + did/do + 主詞 + 原形動詞**。

    `How long did you study last night?` → `Four hours.`  
    👉 How long + did + 主詞 + 原形動詞 …?

*   **規則：** 問「做某事要花多久」用 **How long + does/did + it + take + to + 原形**。

    `How long does it take to cook rice?` → `It takes 20 minutes.`  
    👉 How long + does + it + take + to + 原形動詞 …?

*   **規則：** 回答「花費時間」的黃金句型是 **It takes + (人) + 時間 + to + 原形**。

    `It takes 20 minutes to cook rice.` / `It took Al two hours to drive to work.`  
    👉 It + takes/took + (人) + 時間 + to + 原形動詞

*   **規則：** 用 **How many + 時間單位** 也可問「待多久」，與 **How long** 意思接近。

    `How many days will you be in Hong Kong?` ≈ `How long will you be in Hong Kong?`  
    👉 How many + days + will + 主詞 + be …?

**📌 考試最常錯**
- 花費時間答成 ***It is 20 minutes to …*** — 應 **It takes 20 minutes to …**。
- **How long does it take** 漏 **to** 或後面不是原形（*to cooking*）。
- **How long** 與 **How far** 混淆（距離 vs 時長）。
- **did** 後又寫過去式（*How long did you stayed?*）。

---

## 13. How far（多遠／距離）— Be 與一般動詞問句與答句

*   **規則：** 問「兩地之間有多遠」常用 **How far + Be + it + from … to …**。

    `How far is it from Mumbai to Delhi?` → `725 miles.` / `It's 725 miles.`  
    👉 How far + Be + it + from A + to B …?

*   **規則：** 問「你住得離學校多遠」用 **How far + do/does + 主詞 + live + from …**。

    `How far do you live from school?` → `Four blocks.` / `I live four blocks from school.`  
    👉 How far + 助動詞 + 主詞 + 原形動詞 + from + 地點 …?

*   **規則：** 回答距離可用 **It is + 距離 + from A to B**（**to B from A** 亦可，意思相同）。

    `It is 3,605 miles from Moscow to Beijing.`  
    👉 It + Be + 距離 + from A + to B

*   **規則：** 也可用 **How many + miles/kilometers/blocks + Be + it …** 問距離。

    `How many miles is it …?` / `How many kilometers …?` / `How many blocks …?`  
    👉 How many + 單位 + Be + it …?

**📌 考試最常錯**
- **How far** 與 **How long** 選錯（問距離卻寫 *How long is it from A to B?*）。
- 答距離漏 **from … to …** 或 **It is** 主詞錯（*The distance is from…* 依題型可接受，但考題常考 **It is …**）。
- **How far do you live** 後接錯介係詞（*live from* 可，但 *live far* 結構與題幹不符時易錯）。

---

## 14. 其他 How 的用法 — How old / How much / How many

*   **規則：** 問年齡用 **How old + Be + 主詞**。

    `How old are you?` → `I'm twelve.` / `She is ten years old.`  
    👉 How old + Be + 主詞 …?

*   **規則：** 問價錢用 **How much + Be + 主詞**（主詞常是 this/these）。

    `How much is this?` → `It's 100 dollars.`  
    👉 How much + Be + 主詞 …?

*   **規則：** 問「多少（不可數）」用 **How much + 不可數名詞 + do/does + 主詞 + 原形動詞**。

    `How much water do you need?` → `A little.`  
    👉 How much + 不可數名詞 + 助動詞 + 主詞 + 原形動詞 …?

*   **規則：** 問「多少個（可數）」用 **How many + 複數名詞 + do/does + 主詞 + 原形動詞**。

    `How many apples do you want?` → `Three, please.`  
    👉 How many + 複數名詞 + 助動詞 + 主詞 + 原形動詞 …?

**📌 考試最常錯**
- **How many** 後接**單數名詞**或不可數（*How many water*）— 可數複數用 **How many**，不可數用 **How much**。
- **How much** 與 **How many** 互選錯（價錢／不可數 vs 可數個數）。
- **How old are you?** 答句格式錯（*I have twelve years old.*）— 應 **I'm twelve.** / **I'm twelve years old.**

---

## 15. Who — 搭配 Be 動詞與一般動詞的問句與答句

*   **規則：** 問「那人是誰」用 **Who + Be + 主詞**。

    `Who is that boy?` → `He is my brother.` / `That's John.`  
    👉 Who + Be + 主詞 …?

*   **規則：** 問「你喜歡誰、你打電話給誰」用 **Who + do/does/did + 主詞 + 原形動詞**。

    `Who do you like?` → `I like my teacher.`  
    👉 Who + 助動詞 + 主詞 + 原形動詞 …?

    `Who did you call?` → `I called my mom.`  
    👉 Who + did + 主詞 + 原形動詞 …?

**📌 考試最常錯**
- **Who** 與 **Whose**／**Whom**（若尚未教可略）混淆；與 **Which** 問「哪一位」時題幹未讀清。
- **Who is** 與 **Who are** 主詞單複數一致（*Who are that boy?* ❌）。
- **Who do you …?** 漏助動詞或 **did** 後動詞未原形。

---

## 16. Who 直接當「主詞」

*   **規則：** 當 **Who** 本身就是句子的**主詞**（「誰做了這件事」）時，**後面直接接動詞**，**不要**再加 do/does/did。

    `Who broke the window?` / `Who wants some ice cream?`  
    👉 Who + 動詞（過去式／現在式）…?

*   **規則：** **Who** 當主詞時，**不可**寫成 **Who did break …**。

    ❌ `Who did break the window?`（不要這樣說）  
    👉 錯誤：Who + did + 原形（多餘的助動詞）

**📌 考試最常錯**
- **Who** 當主詞卻多加 **did/does/do**（*Who did break…?*、*Who does live…?*）。
- 動詞與 **Who** 單複數概念搞混（*Who want* vs *Who wants* — 意義上常當單數處理）。

---

## 17. Why — 搭配 Be 動詞與一般動詞的問句與答句

*   **規則：** 問「為什麼（處於某狀態）」用 **Why + Be + 主詞 + …**。

    `Why are you late?` → `Because the bus was late.` / `Because I overslept.`  
    👉 Why + Be + 主詞 …?／答：Because + 子句

*   **規則：** 問「為什麼（做某動作）」用 **Why + do/does + 主詞 + 原形動詞**。

    `Why do you cry?` → `Because I am sad.`  
    👉 Why + 助動詞 + 主詞 + 原形動詞 …?

*   **規則：** 口語也可用 **Because of + 名詞** 說明原因。

    `Because of the rain.`  
    👉 Because of + 名詞

**📌 考試最常錯**
- 問句缺 **do/does**（*Why you cry?*）— 應 **Why do you cry?**
- 答句用 **Because** 卻接**片語**不完整（*Because late.*）— 應 **Because I am late.** 或 **Because of the traffic.**
- **Why** 與 **Because** 對題時寫成 *Why … Because …* 同一句疊用（依題型分兩句較清楚）。

---

## 18. How about / What about（提議與反問）+ 介係詞留在句尾

*   **規則：** **How about / What about** 後面接**名詞或代名詞**，用來提議。

    `How about three o'clock?`  
    👉 How about + 名詞片語 …?

*   **規則：** 後面若要接動作，用 **V-ing（動名詞）**。

    `How about going to the zoo?` / `What about asking Sally over for dinner?`  
    👉 How about／What about + V-ing …?

*   **規則：** **How about you?** 用來把剛才的話題丟回對方（「你呢？」）。

    `I'm tired. How about you?`（你呢？＝ Are you tired?）  
    👉 陳述句 + How about you?

*   **規則：** 動詞與介係詞是固定搭配（如 **come from**, **look for**）時，在疑問句裡介係詞常留在**句尾**。

    `Where do you come from?`  
    👉 疑問詞 + 助動詞 + 主詞 + 動詞 + 介係詞（句尾）

    `What are you looking for?`  
    👉 疑問詞 + Be + 主詞 + V-ing + 介係詞（句尾）

**📌 考試最常錯**
- **How about go …** ❌ — 應 **How about going …?**（介詞片語後接 **V-ing**）。
- 介係詞提到**主詞前**寫成 *From where do you come?*（較正式或較少考初階）— 課內常考**句尾介係詞**。
- **What are you looking?** 漏 **for**。

---

## 19. Be 動詞與助動詞的 Tag Questions（附加問句）

*   **規則：** 附加問句重複**前面直述句的 Be 動詞**，且常遵守「前肯後否、前否後肯」。

    `Jill is sick, isn't she?`  
    👉 肯定直述 + Be + 主詞代名詞 + 否定簡問?

    `Marie isn't from Athens, is she?`  
    👉 否定直述 + Be + 主詞代名詞 + 肯定簡問?

*   **規則：** **There is** 的附加問句用 **isn't there?**。

    `There's enough time, isn't there?`  
    👉 There + Be … + isn't there?

*   **規則：** **I'm** 的附加問句否定用 **am I?**；若前面是**肯定 I'm …**，附加問句常用 **aren't I?**（慣用法）。

    `I'm not late, am I?` / `I'm late, aren't I?`  
    👉 I'm + not …, am I?／I'm …, aren't I?

*   **規則：** 前面直述句用**一般動詞**時，附加問句用 **do/does/did** 的簡短形式。

    `You know Bill, don't you?`  
    👉 肯定直述 + don't + 主詞?

    `You don't know Tom, do you?`  
    👉 否定直述 + do + 主詞?

*   **規則：** **上揚語調** 較像「確認資訊」；**下降語調** 較像「尋求認同」。

    *   **上揚 ⤴**：較不確定，想確認對方是否同意。
    *   **下降 ⤵**：較確定，想尋求認同。

**📌 考試最常錯**
- **前肯後否／前否後肯**搞反（*He is tall, isn't he?* vs *is he?* 混用）。
- **I'm …** 肯定後附加問句寫成 *amn't I?* — 慣用 **aren't I?**；否定用 **am I?**。
- **There is …** 的 tag 寫成 *isn't it?* — 應 **isn't there?**
- 一般動詞句用錯助動詞（*You like it, isn't you?*）— 應 **don't you?**

## 20. 考前檢查表：你中了幾個常犯錯誤？

**📌 考試最常錯（本節＝綜合複習）**  
以下四點為全卷最高頻錯誤，與單元 1 倒裝公式直接對應；寫任何 Wh- 問句皆可依此快篩。

寫完疑問句後，可用下面四點自我檢查（每點皆為「錯誤例 → 正確例」）：

1. **有沒有漏掉助動詞或 Be 動詞？**

    **規則：** 一般動詞的疑問句**一定要有** do/does/did；有 Be 時**一定要有** Be。

    ❌ `Where you live?` → ⭕ `Where do you live?`  
    👉 Where + **do** + 主詞 + 原形動詞 …?

2. **順序對了嗎？（Be／助動詞有沒有移到主詞前）**

    **規則：** 疑問句裡 **Be 或助動詞** 必須在**主詞前面**。

    ❌ `What you are doing?` → ⭕ `What are you doing?`  
    👉 What + **are** + 主詞 + V-ing …?

3. **一般動詞有沒有還原成原形？**（did / does / do 後面要原形）

    **規則：** **助動詞後**的主要動詞用**原形**。

    ❌ `Where did you went?` → ⭕ `Where did you go?`  
    👉 did + 主詞 + **原形** go

    ❌ `What does he wants?` → ⭕ `What does he want?`  
    👉 does + 主詞 + **原形** want

4. **有沒有誤用 Be 動詞取代助動詞？**（live 等一般動詞要用 do/does）

    **規則：** **live** 等一般動詞**不能**和 **are you** 直接連用成問句。

    ❌ `Where are you live?` → ⭕ `Where do you live?`  
    👉 Where + **do** + 主詞 + live …?
