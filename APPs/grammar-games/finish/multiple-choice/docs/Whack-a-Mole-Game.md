# 打地鼠文法選擇題 — 總覽

**程式**：`APPs/grammar-games/finish/multiple-choice/`  
**完整規格**：[`Fast-MultipleChoice-Game.md`](./Fast-MultipleChoice-Game.md)（含時態、WH 題庫對照表）

## 玩法摘要

- **6 洞（3×2）**，最多 **3 隻地鼠** 輪流舉牌（A～D 隨機）
- 練習 **15 分鐘**；WH / 時態入口見文法大廳
- 答對自動下一題；錯 3 次顯示解析

## 文法大廳入口

| 分頁 | 遊戲 |
|------|------|
| Time Tenses | Present / Past / Future / Present Continuous 各一個打地鼠 |
| Wh- Questions | How 系列三份題庫（9–10、11–12、14 單元） |

詳見 `Fast-MultipleChoice-Game.md` §4。
